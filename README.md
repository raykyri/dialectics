### Agent Dialectics

Working folder for the [Electric Monks](https://github.com/KyleAMathews/hegelian-dialectic-skill) multi-turn AI dialectics system.

### How to use

```
git clone https://github.com/raykyri/dialectics.git
cd dialectics
claude
```

Claude will ask you to trust this directory. After selecting yes, run:

```
/hegelian-dialectic-skill
```

Follow the instructions returned by the skill.
